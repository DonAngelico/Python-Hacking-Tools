🛡️ Python Cybersecurity Toolkit

A curated collection of Python-based tools demonstrating practical cybersecurity techniques, including network attacks, credential testing, and system monitoring.

This project highlights hands-on experience with penetration testing concepts, scripting, and security automation.

⚠️ Ethical Use Only: All tools are intended strictly for authorized environments such as labs, CTFs, and educational use.

💼 Project Overview

This repository was built to strengthen skills in:

🔐 Offensive Security Techniques
🌐 Network Traffic Analysis
⚡ Python Automation for Security
🧠 Understanding Real-World Attack Vectors

Each tool reflects a commonly studied concept in cybersecurity and demonstrates the ability to implement it programmatically.

🧰 Tools & Skills Demonstrated
🔎 Network Security
Vulnerability Scanner (Vul_scanner)
→ Identifies potential weaknesses in target systems
ARP Spoofer (arpspoofer)
→ Performs man-in-the-middle (MITM) attack simulations

Skills: Packet manipulation, networking protocols, attack simulation

🔐 Access & Authentication Testing
SSH Brute Force Tool (sshbruteforce)
→ Tests password strength against SSH services
Password Sniffer (passwordsniffer)
→ Captures plaintext credentials over insecure channels

Skills: Authentication security, brute-force logic, traffic inspection

🎯 System Monitoring & Exploitation Concepts
Keylogger (keylogger)
→ Captures keystrokes for monitoring user activity
Camera Access Script (camera.py)
→ Demonstrates interaction with system hardware

Skills: OS interaction, input capture, security implications of endpoints

🌐 Web & API Security
API Fuzzer (apifuzzer)
→ Tests endpoints for unexpected behavior and vulnerabilities

Skills: Input fuzzing, API testing, automation

📧 Additional Tools
Email Bomb Script (email_bomb.py)
→ Simulates email flooding for stress-testing systems
Wordlist File (small.txt)
→ Used for brute-force and fuzzing operations
🚀 Technical Highlights

Covers multiple domains:
Network attacks

API security testing
🧪 How to Run
git clone https://github.com/DonAngelico/Python-Hacking-Tools.git
cd Python-Hacking-Tools

Run a tool:

python camera.py

Or:

cd sshbruteforce
python sshbruteforce.py

Some scripts may require administrator/root privileges.
